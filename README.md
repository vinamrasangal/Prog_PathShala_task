## Frontend Modern Design Website
  TypeScript, Next.js, Tailwind CSS, and Shadcn UI.

## Features:
- Dark/Light Mode
- Sliding Carousel
- Faster Image Loads
- Displaying the Numbers Dynamically

## Start the App Using Commands:
1. `npm i`
2. `npm run dev`
